Developer Notes

This implementation uses the "Checkbox Hack" (input[type="checkbox"] + label) to demonstrate mobile menu toggling without JavaScript. In a production React/Next.js environment, this should be replaced with state management (e.g., useState) to toggle the .navbar__nav class for accessibility reasons.

Key CSS Classes:

  • .navbar - Base container
  • .navbar--sticky - Applies position: sticky
  • .navbar--admin - Applies dark theme tokens
  • .navbar__link--active - Visual indicator for current page

Accessibility Checklist:

  • Ensure aria-current="page" is applied to the active link programmatically.
  • Ensure hamburger menu button has aria-expanded state managed via JS.
  • Ensure focus is trapped within the mobile menu when opened on small screens.
