Component Name

Digital Warranty Card

Component Type

organism

Description

A comprehensive card component displaying product warranty status, key details (serial, dates), and primary actions. Acts as the central hub for product management.

Design System Alignment

Utilizes Card, Status Badge, and Button molecules. Consumes color semantic tokens for status states and spacing tokens for internal layout.

Design Tokens Used

--color-primary-600
--radius-lg
--shadow-md
--color-success-bg
--color-warning-bg
--color-error-bg

Responsive Specifications

Mobile Adaptations

  • Header elements stack vertically
  • Action buttons take full width
  • Meta data stacks for readability
  • Minimum touch targets of 44px maintained

Tablet Adaptations

  • Grid layout for dates
  • Buttons share rows

Desktop Enhancements

  • Hover effects on card and buttons
  • Horizontal layout for meta data
  • Tooltip visibility on status badges

Accessibility Features

Keyboard Support

Logical tab order through actions

Screen Reader Support

Status badges use hidden text for context, semantic article tags used for card structure

Focus Management

Visible focus ring on all interactive elements

Aria Attributes

  • aria-labelledby
  • aria-label for icon buttons
  • role='article'

Color Contrast

Status text meets 4.5:1 ratio against badge background

Implementation Notes

Embedded Css Architecture

BEM naming convention, scoped CSS variables for theming

Component Organization

Header (Status), Body (Product), Dates (Grid), Footer (Actions)

Responsive Design

Container queries simulated via flex/grid wrapping logic

Accessibility Compliance

Native button elements, proper heading hierarchy

Usage Guidelines

When To Use

On the main dashboard list or detailed product view page

Content Guidelines

Dates should be localized. Serial numbers should be monospace.

Placement Rules

Grid on desktop, Stack on mobile

Customization Options

  • Add .warranty-card--active/expired/expiring classes to parent
  • Toggle loading state with .warranty-card--loading

Testing Recommendations
