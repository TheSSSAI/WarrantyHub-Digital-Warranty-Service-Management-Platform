Description

A comprehensive data grid component supporting sorting, filtering, row selection, pagination, and responsive layout adaptations. Designed for high-density information display with mobile-responsive card transformation.

Design System Alignment

Utilizes primary colors for interactive elements, neutral palette for structure, and semantic tokens for status indicators. Implements the system's spacing scale and typography hierarchy.

Design Tokens Used

--color-white
--color-gray-200
--color-gray-50
--color-primary-50
--radius-lg
--spacing-4
--font-size-sm

Responsive Specifications

Mobile Adaptations

  • Table header is hidden visually
  • Rows transform into individual cards
  • Cells become flex containers with data-label pseudo-elements
  • Actions move to bottom of card or accessible menu
  • Checkbox selection moves to card header

Tablet Adaptations

  • Horizontal scrolling enabled for wide tables
  • Reduced padding in cells

Desktop Enhancements

  • Hover effects on rows
  • Sticky headers for long lists
  • Expanded filters in toolbar

Accessibility Features

Keyboard Support

Sort headers focusable, Rows navigable, Action buttons accessible via Tab

Screen Reader Support

aria-sort on headers, scope attributes, proper table semantics preserved even in CSS grid mode

Focus Management

Visible focus ring within cells and on sort triggers

Aria Attributes

  • aria-sort='ascending/descending'
  • aria-label='Select row'
  • aria-selected='true/false'

Color Contrast

Text and borders meet 4.5:1 ratio
