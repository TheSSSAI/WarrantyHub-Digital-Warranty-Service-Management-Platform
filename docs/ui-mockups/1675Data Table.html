Usage Guidelines

  • Use the Comprehensive Data Table for main dashboard views where sorting and actions are required.
  • On mobile devices (< 768px), the table automatically transforms into a Card View. Data labels are generated via CSS content property.
  • Always include Pagination for datasets larger than 20 rows to ensure performance.
  • Use the Skeleton loading state during API fetches to prevent layout shift.
  • Ensure aria-label is populated for all icon-only buttons for accessibility.
