Responsive Specifications

Mobile Adaptations

  • Stepper collapses to text-based progress indicator
  • Form grid becomes single column
  • Action buttons stack vertically or fix to bottom
  • Upload zone becomes compact touch target

Tablet Adaptations

  • Stepper shows icons without labels
  • Form uses 2-column grid

Desktop Enhancements

  • Full horizontal stepper with labels
  • Side-by-side layout for OCR preview and form data
  • Expanded drag-and-drop zone
