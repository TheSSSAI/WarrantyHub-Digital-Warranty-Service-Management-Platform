Usage Guidelines

When To Use

Use KPI widgets for top-level dashboard metrics. Use Search bar for lists > 20 items.

Content Guidelines

KPI labels should be short (1-3 words). Notification text should be truncated after 2 lines.

Placement Rules

Search bar always at top of list views. KPI widgets at top of dashboards.

Customization Options

  • KPI: .kpi-widget--trend-up / .kpi-widget--trend-down
  • Notification: .notification-item--unread
  • Document: .doc-card--invoice / .doc-card--warranty
